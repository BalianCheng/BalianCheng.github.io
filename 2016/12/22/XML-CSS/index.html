在XML里用到过的CSS的一点点基础内容

CSS基础


CSS的定义语法由三部分构成：选择器(selector)，属性(property)，属性值(value)

CSS可以设置多个元素的选择器：

selector1、selector2、selector3公用同一种样式
类(class)选择器可以为同名的元素设置不同的样式：

标识(ID)选择器：唯一标识一个元素

CSS属性


display常用属性

属性值 说明
none 不显示
block 块级元素，换行与别的元素分开
inline 默认，同一行显示
list-item 作为列表显示

font常用属性

属性值 说明
font-family 字体
font-size 尺寸
font-style 字形样式
font-variant 大小写
font-weight 粗细

CSS可以使用color指定元素字体颜色

CSS允许添加边框

属性值 说明
none 无边框
dotted 点状边框
dashed 虚线
solid 实线
double 双线
groove 3D 凹槽边框
ridge 3D 垄状边框
inset 3D inset 边框
outset 3D outset 边框
inherit 从父元素继承

XML背景属性

属性值 说明
background-color 背景颜色
background-position 背景图像的位置
background-size 规定背景图片的尺寸
background-repeat 规定如何重复背景图像
background-origin 背景图片的定位区域
background-clip 规定背景的绘制区域
background-attachment 规定背景图像是否固定
background-image 背景图像
inherit 从父元素继承

更多属性参考CSS参考手册

链接样式：在xml中加入<?xml-stylesheet type="text/css" href="样式名.css"?>
或者@import url(stylesheetURL)
无需链接的内部CSS
